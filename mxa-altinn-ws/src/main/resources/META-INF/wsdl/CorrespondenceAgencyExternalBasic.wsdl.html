      2. InsertCorrespondenceBasicV2
        Source code
        Altinn dokumentasjon:


        Summary: This operation is used to send a correspondence to a user.
        Remarks: This operation is versioned as V2 as it contain some additional information. Prefered over InsertCorrespondenceBasic. Optional additional information include CaseId used if the correspondence(s) shall be a part of a collaboration service and AllowForwarding in order to specify whether the correspondence(s) has the ability to be forwarded as email or not by the end user in SBL. Note thet when not submitting the AllowForwarding element it is default that the correspondence(s) can be forwarded.
        Parameter:"systemUserName" System user name is the system identifikator for the end user system as registered in Altinn - mandatory parameter
        Parameter:"systemPassword" System password is the password for the registered end user system - mandatory parameter
